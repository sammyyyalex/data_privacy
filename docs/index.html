Our Mission

With all the exciting aspects of the internet, it is easy to ignore online safety while browsing; our online data is always being collected, just from the websites we visit and social media we use. We created this website to inform and introduce these topics to others. The internet is being used by almost everyone nowadays, but not a lot are aware of the privacy issues that exist. It is essential for any internet user to be aware of these issues in order to safely use the internet.
